**fdp-conducted:**

If any adapter has truly unique rendering that can't be driven by the schema, keep that specific rendering but delegate everything else to BaseEntryAdapter:

```typescript

    <BaseEntryAdapter

        return null; // use default rendering

  );
}
```
