### Step 5: Handle client vs server split

The page itself is a server component (async, fetches data). Sections that need interactivity (form inputs, file uploads, toggles) must be client components ("use client"). Sections that just display data can stay as server components.

**Rule: Only add "use client" to components that actually need it (onClick, useState, useEffect, etc.)**
